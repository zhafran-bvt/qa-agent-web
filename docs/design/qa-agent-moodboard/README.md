# QA Agent Moodboard

Design references for the QA Agent workbench UI. These images define the visual direction for an internal QA daily workbench: dense operational layout, persistent workflow navigation, trust/diagnostic surfaces, review tables, approval gates, and TestRail push readiness.

## Tiles

- `01-operational-workbench.png` - Overall daily workbench layout and navigation direction.
- `02-scope-snapshot-trust.png` - Scope Snapshot, source evidence, and confidence/trust presentation.
- `03-bdd-review-workspace.png` - BDD review table and selected-case detail workspace.
- `04-setup-health-diagnostics.png` - Integration readiness, setup health, and diagnostics direction.
- `05-approval-push-gates.png` - Approval gates, duplicate review, selected cases, and TestRail push panel.
- `05-approval-push-gates-mockup.html` - Static HTML recreation of the approval/push gates reference.
- `06-workflow-history-audit.png` - Workflow history and audit trail direction.
